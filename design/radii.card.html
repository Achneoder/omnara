Tab switcher. `underline` for primary page sections, `pill` for compact in-card filters. Controlled via `value`/`onChange`. ```jsx const [tab, setTab] = React.useState("review"); }, { value: "live", label: "Live" }, ]} /> ``` Props: `items` (`{value,label,icon?,count?}`), `value`, `onChange`, `variant (underline|pill)`.
